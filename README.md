# TranslateMyCall

TranslateMyCall is a phone-first AI interpreter. You can try it at [translatemycall.com](https://translatemycall.com) or by calling [(228) 888-8666](tel:+12288888666).

Demo video: [https://youtu.be/c5GA44FGk_U](https://youtu.be/c5GA44FGk_U)

## What is this?

TranslateMyCall lets you dial a phone number and get real-time translation for phone calls or in-person speakerphone conversations. It is a v2 of an earlier project we built about two years ago, updated with lessons from building the ThunderPhone call automation platform.

## What was new during the hackathon?

During the hackathon, we built the onboarding phone agent and the translation experience. We also launched the public landing page and phone number.

The project builds on existing ThunderPhone infrastructure, with a lot of deployment, telephony, and call automation infrastructure copied over from ThunderPhone.
