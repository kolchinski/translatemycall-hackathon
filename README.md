# TranslateMyCall

TranslateMyCall is a phone-first AI interpreter. You can try it at [translatemycall.com](https://translatemycall.com) or by calling [(228) 888-8666](tel:+12288888666).

Demo video: [https://youtu.be/c5GA44FGk_U](https://youtu.be/c5GA44FGk_U)

## What is this?

TranslateMyCall lets you dial a phone number and get real-time translation for phone calls or in-person speakerphone conversations. It is a v2 of an earlier project we built about two years ago, updated with lessons from building the ThunderPhone call automation platform.

## What was new during the hackathon?

During the hackathon, we built the onboarding phone agent and the translation experience. We also launched the public landing page and phone number.

The project builds on existing ThunderPhone infrastructure, with a lot of deployment, telephony, and call automation infrastructure copied over from ThunderPhone.

## Hackathon theme fit

TranslateMyCall is a voice AI project focused on real-time phone translation. The hackathon work centered on making the phone onboarding and translation flow usable end-to-end.

We did not use every sponsor tool. The parts most aligned with the hackathon themes are voice, agent behavior, and improving the reliability of a phone-based AI experience.

## Tool feedback

The biggest lesson from this build was that the product experience depends heavily on the phone agent being simple, fast, and predictable. Translation quality matters, but the surrounding call flow matters just as much: callers need clear prompts, quick setup, and no app install.

We would like more evaluation tooling specifically for voice-agent latency, turn-taking, language selection, and recovery from confused callers. Those are the areas where self-improvement loops would be most valuable for this kind of product.
